1. Pengenalan SATRIA

1.1 Apa itu SATRIA?

SATRIA (Sistem Absensi Trimulia Jakarta) adalah aplikasi berbasis web yang dirancang khusus untuk mengelola absensi siswa di SMK Trimulia Jakarta. Sistem ini menggunakan teknologi GPS untuk memastikan siswa melakukan absensi dari lokasi sekolah.

1.2 Fitur Utama

  • Absensi Berbasis GPS: Siswa hanya bisa absen jika berada dalam radius sekolah
  • Absensi Datang & Pulang: Sistem mencatat waktu kedatangan dan kepulangan siswa
  • Pengajuan Izin/Sakit: Siswa dapat mengajukan izin atau sakit dengan bukti pendukung
  • Rekap Otomatis: Sistem menghasilkan laporan absensi harian, mingguan, dan bulanan
  • Multi Role: Mendukung 4 jenis pengguna dengan hak akses berbeda

1.3 Jenis Pengguna

Role Deskripsi Hak Akses Utama
Kepala Sekolan dan Kesiswaan Sebagai Admin Pengelola sistem secara keseluruhan Kelola data sekolah, lihat rekap global, atur pengaturan sistem
Guru Piket Guru yang bertugas mengawasi absensi harian Kelola absensi harian, verifikasi izin/sakit, buat rencana absensi
Wali Kelas Guru yang bertanggung jawab atas satu kelas Lihat rekap absensi kelas, pantau kehadiran siswa di kelasnya
Siswa Pengguna yang melakukan absensi Absen datang/pulang, ajukan izin/sakit, lihat riwayat absensi

3. Panduan untuk Admin/Kepala Sekolah

3.1 Dashboard Admin

Dashboard menampilkan ringkasan statistik absensi seluruh sekolah:

  • Total siswa
  • Jumlah siswa hadir hari ini
  • Jumlah siswa terlambat, izin, sakit, alfa, dan belum hadir
  • Persentase kehadiran
  • Grafik tren kehadiran
  • Aktivitas terbaru di sistem

3.2 Menu Kelola Sekolah

Menu ini berisi berbagai fitur untuk mengelola data sekolah:

3.2.1 Kelola Jurusan

Fungsi: Mengelola data jurusan yang ada di sekolah

Cara Menggunakan:

  1. Klik menu "Kelola Sekolah" ‚Üí "Jurusan"
  2. Untuk menambah jurusan baru:
    • Klik tombol "Tambah Jurusan"
    • Masukkan nama jurusan (contoh: Teknik Komputer dan Jaringan)
    • Klik "Simpan"
  3. Untuk mengedit jurusan:
    • Klik tombol "Edit" pada jurusan yang ingin diubah
    • Ubah nama jurusan
    • Klik "Simpan"
  4. Untuk menghapus jurusan:
    • Klik tombol "Hapus"
    • Konfirmasi penghapusan
‚ö†Ô∏è Perhatian: Menghapus jurusan akan menghapus semua kelas dan riwayat kelas yang terkait!

3.2.2 Kelola Kelas

Fungsi: Mengelola data kelas (tingkat, jurusan, paralel)

Cara Menggunakan:

  1. Klik menu "Kelola Sekolah" ‚Üí "Kelas"
  2. Untuk menambah kelas baru:
    • Klik tombol "Tambah Kelas"
    • Pilih tingkat (X, XI, atau XII)
    • Pilih jurusan
    • Masukkan paralel (1, 2, 3, dll) atau bisa juga dikosongkan
    • Klik "Simpan"
  3. Untuk mengedit atau menghapus kelas, gunakan tombol yang tersedia

3.2.3 Kelola Wali Kelas

Fungsi: Menugaskan guru sebagai wali kelas

Cara Menggunakan:

  1. Klik menu "Kelola Sekolah" ‚Üí "Wali Kelas"
  2. Untuk menambah wali kelas:
    • Klik tombol "Tambah Wali Kelas"
    • Masukkan username dan nama
    • Masukkan username dan password untuk login
    • Pilih kelas yang akan diampu
    • Klik "Simpan"

3.2.4 Kelola Guru Piket

Fungsi: Mengelola data guru piket

Cara Menggunakan:

  1. Klik menu "Kelola Sekolah" ‚Üí "Guru Piket"
  2. Untuk menambah guru piket:
    • Klik tombol "Tambah Guru Piket"
    • Masukkan nama guru
    • Masukkan username dan password
    • Klik "Simpan"

3.2.5 Jadwal Piket

Fungsi: Mengatur jadwal piket guru per hari

Cara Menggunakan:

  1. Klik menu "Kelola Sekolah" ‚Üí "Jadwal Piket"
  2. Pilih hari yang ingin diatur
  3. Pilih guru piket yang bertugas
  4. Klik "Simpan"

3.2.6 Import Data Siswa

Fungsi: Mengimpor data siswa dalam jumlah banyak menggunakan file Excel

Cara Menggunakan:

  1. Klik menu "Kelola Sekolah" ‚Üí "Import Data Siswa"
  2. Download template Excel yang disediakan
  3. Isi data siswa sesuai format template:
    • NIS
    • Nama
    • Kelas ID
    • Username
    • Password
  4. Upload file Excel yang sudah diisi
  5. Klik "Import"
  6. Sistem akan memproses dan menampilkan hasil import
üí° Tips: Pastikan format file Excel sesuai dengan template untuk menghindari error saat import.

3.2.7 Kelola Data Siswa

Fungsi: Mengelola data siswa secara individual

Cara Menggunakan:

  1. Klik menu "Kelola Sekolah" ‚Üí "Kelola Data Siswa"
  2. Gunakan fitur pencarian untuk menemukan siswa
  3. Untuk menambah siswa baru, klik "Tambah Siswa"
  4. Untuk mengedit data siswa, klik tombol "Edit"
  5. Untuk menghapus siswa, klik tombol "Hapus"

3.2.8 Riwayat Kelas

Fungsi: Melihat riwayat perpindahan kelas siswa

Cara Menggunakan:

  1. Klik menu "Kelola Sekolah" ‚Üí "Riwayat Kelas"
  2. Sistem akan menampilkan daftar riwayat kelas siswa
  3. Gunakan filter untuk mencari riwayat tertentu

3.3 Rekap Absensi

Fungsi: Melihat dan mengekspor rekap absensi

Cara Menggunakan:

  1. Klik menu "Rekap"
  2. Pilih periode rekap:
    • Harian: Rekap per hari
    • Mingguan: Rekap per minggu
    • Bulanan: Rekap per bulan
  3. Pilih tanggal yang diinginkan
  4. Filter berdasarkan kelas (opsional)
  5. Untuk mengekspor:
    • Klik "Export Excel" untuk format Excel
    • Klik "Export PDF" untuk format PDF
üí° Tips: Rekap mingguan dan bulanan akan mengagregasi data dari beberapa hari secara otomatis.

3.4 Pengaturan Sistem

Fungsi: Mengatur konfigurasi sistem absensi

Cara Menggunakan:

  1. Klik menu "Pengaturan Sistem"
  2. Atur lokasi sekolah:
    • Klik pada peta untuk memilih lokasi
    • Atau gunakan tombol "Gunakan Lokasi Saat Ini"
    • Atau masukkan koordinat latitude dan longitude secara manual
  3. Atur radius absensi (dalam meter)
  4. Atur jam masuk dan jam pulang
  5. Atur toleransi keterlambatan (dalam menit)
  6. Klik "Simpan"
‚ö†Ô∏è Perhatian: Pengaturan ini akan mempengaruhi seluruh sistem absensi. Pastikan koordinat lokasi dan radius sudah benar sebelum menyimpan.

Fungsi: Melihat dan mengelola absensi siswa hari ini

Cara Menggunakan:

  1. Klik menu "Absensi" ‚Üí "Absen Hari Ini"
  2. Sistem menampilkan daftar siswa yang sudah absen hari ini

6.3 Absen Datang

Fungsi: Mencatat kehadiran saat tiba di sekolah

Cara Menggunakan:

  1. Pastikan GPS HP Anda aktif
  2. Pastikan Anda berada di area sekolah (dalam radius yang ditentukan)
  3. Klik tombol "Absen Datang" di halaman home
  4. Sistem akan mengambil lokasi GPS Anda (tunggu beberapa saat)
  5. Peta akan menampilkan:
    • Lokasi Anda (ikon orang)
    • Lokasi sekolah (ikon sekolah)
    • Lingkaran radius absensi
    • Garis merah putus-putus menunjukkan jarak Anda ke sekolah
  6. Sistem akan menampilkan informasi jarak Anda dari sekolah
  7. Jika Anda berada dalam radius, klik tombol "Absen Datang"
  8. Konfirmasi absensi
  9. Sistem akan mencatat waktu kedatangan Anda
‚úÖ Status Kehadiran:
  • Hadir: Jika Anda absen sebelum batas jam masuk
  • Terlambat: Jika Anda absen setelah batas jam masuk + toleransi
‚ö†Ô∏è Perhatian:
  • Pastikan GPS HP Anda aktif dan akurat
  • Jika lokasi tidak akurat, coba refresh GPS dengan tombol "Refresh GPS"
  • Anda hanya bisa absen jika berada dalam radius sekolah
  • Jika Anda berada di luar radius, sistem akan menampilkan peringatan dan jarak yang harus Anda dekati

6.4 Absen Pulang

üí° Tips: Anda hanya bisa absen pulang setelah jam pulang yang ditentukan sekolah.

6.5 Pengajuan Izin / Sakit

Fungsi: Mengajukan izin atau sakit jika tidak bisa hadir ke sekolah

Cara Menggunakan:

  1. Klik tombol "Izin / Sakit" di halaman home
  2. Pilih jenis pengajuan:

    • Sakit: Untuk kondisi sakit
  3. Tanggal akan otomatis terisi dengan hari ini
  4. Masukkan alasan/keterangan (minimal 10 karakter)
  5. Upload bukti (opsional):
    • Untuk izin: surat izin, undangan, dll
    • Untuk sakit: surat dokter, foto obat, dll

  6. Klik "Kirim Pengajuan"
  7. Tunggu verifikasi dari guru piket
üí° Tips:
  • Ajukan izin/sakit sedini mungkin
  • Sertakan bukti yang jelas untuk mempercepat verifikasi
  • Tulis keterangan dengan lengkap dan jelas

7. FAQ dan Troubleshooting

7.1 Pertanyaan Umum

Q: Bagaimana jika saya lupa password?

A: Hubungi admin sekolah untuk melakukan reset password. Admin dapat mengatur password baru untuk akun Anda.

Q: Kenapa saya tidak bisa absen padahal sudah di sekolah?

A: Beberapa kemungkinan penyebab:

  • GPS HP Anda tidak aktif - Aktifkan GPS di pengaturan HP
  • Lokasi GPS tidak akurat - Coba refresh GPS atau tunggu beberapa saat
  • Anda berada di luar radius sekolah - Pastikan Anda berada dalam area yang ditentukan
  • Browser tidak mendapat izin akses lokasi - Izinkan akses lokasi di browser

Q: Apakah saya bisa absen menggunakan laptop/PC?

A: Bisa, tetapi tidak disarankan karena laptop/PC biasanya tidak memiliki GPS yang akurat. Gunakan smartphone untuk hasil terbaik.

Q: Bagaimana jika HP saya rusak/hilang saat harus absen?

A: Lapor ke guru piket yang bertugas. Guru piket dapat melakukan absensi manual untuk Anda.

Q: Berapa lama waktu yang dibutuhkan untuk verifikasi izin/sakit?

A: Verifikasi dilakukan oleh guru piket yang bertugas pada hari itu. Biasanya diproses dalam 1-2 jam kerja.

Q: Apakah saya bisa mengubah data absensi yang sudah tercatat?

A: Tidak. Siswa tidak bisa mengubah data absensi. Jika ada kesalahan, hubungi guru piket atau wali kelas Anda.

Q: Bagaimana cara melihat rekap absensi bulanan saya?

A: Klik menu "Riwayat" di aplikasi. Anda dapat melihat semua riwayat absensi Anda di sana.

7.2 Troubleshooting

Masalah: GPS tidak bisa mengambil lokasi

Solusi:

  1. Pastikan GPS HP Anda aktif (cek di Settings ‚Üí Location)
  2. Pastikan browser mendapat izin akses lokasi:
    • Chrome: Klik ikon gembok di address bar ‚Üí Site settings ‚Üí Location ‚Üí Allow
    • Firefox: Klik ikon (i) di address bar ‚Üí Permissions ‚Üí Location ‚Üí Allow
  3. Coba refresh halaman
  4. Jika masih gagal, coba gunakan browser lain
  5. Pastikan Anda mengakses via HTTPS (bukan HTTP)

Masalah: Lokasi GPS tidak akurat

Solusi:

  1. Keluar dari gedung dan coba di area terbuka
  2. Tunggu beberapa saat agar GPS mendapat sinyal yang lebih baik
  3. Klik tombol "Refresh GPS" untuk memperbarui lokasi
  4. Restart HP Anda
  5. Pastikan mode lokasi di HP diset ke "High Accuracy"

Masalah: Tombol absen tidak bisa diklik

Solusi:

  1. Pastikan Anda berada dalam radius sekolah
  2. Pastikan GPS sudah berhasil mengambil lokasi Anda
  3. Cek apakah Anda sudah absen hari ini (tidak bisa absen datang 2x dalam sehari)
  4. Refresh halaman dan coba lagi

Masalah: Upload bukti izin/sakit gagal

Solusi:

  1. Pastikan ukuran file tidak lebih dari 5MB
  2. Pastikan format file adalah JPG, PNG, atau PDF
  3. Coba kompres gambar terlebih dahulu jika terlalu besar
  4. Pastikan koneksi internet stabil

Masalah: Tidak bisa login

Solusi:

  1. Pastikan username dan password benar (perhatikan huruf besar/kecil)
  2. Pastikan tidak ada spasi di awal atau akhir username/password
  3. Coba refresh halaman
  4. Clear cache dan cookies browser
  5. Jika masih gagal, hubungi admin untuk reset password

Masalah: Halaman tidak muncul dengan benar

Solusi:

  1. Clear cache browser (Ctrl+Shift+Delete)
  2. Refresh halaman (Ctrl+R atau F5)
  3. Coba gunakan browser lain (Chrome, Firefox, Edge)
  4. Pastikan browser Anda versi terbaru

7.3 Tips Penggunaan

Untuk Siswa:

  • Selalu aktifkan GPS sebelum berangkat ke sekolah
  • Absen segera setelah tiba di sekolah
  • Jangan lupa absen pulang sebelum meninggalkan sekolah
  • Ajukan izin/sakit sedini mungkin jika tidak bisa hadir
  • Simpan bukti izin/sakit dalam format digital untuk memudahkan upload

Untuk Guru Piket:

  • Cek daftar pengajuan izin/sakit secara berkala
  • Verifikasi bukti yang dilampirkan sebelum menyetujui
  • Gunakan fitur absensi manual untuk siswa yang bermasalah dengan HP
  • Monitor daftar siswa yang belum hadir untuk tindak lanjut
  • Gunakan filter untuk mempermudah pencarian data

Untuk Wali Kelas:

  • Pantau kehadiran siswa di kelas Anda setiap hari
  • Hubungi siswa yang sering tidak hadir atau terlambat
  • Buat rekap bulanan untuk laporan ke orang tua
  • Koordinasi dengan guru piket jika ada masalah absensi

Untuk Admin:

  • Backup data secara berkala
  • Pastikan pengaturan lokasi dan radius sudah benar
  • Monitor aktivitas sistem secara rutin
  • Update data siswa dan kelas setiap awal tahun ajaran
  • Berikan pelatihan kepada pengguna baru
