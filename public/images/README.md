# Preview Images

## Required Image

Place the template preview image at:
`/public/images/preview.png`

This image should show a screenshot of the Excel template with the following columns:
- NIS
- Nama  
- Jenis Kelamin
- Kelas

## Image Requirements

- **Format**: PNG (recommended) or JPG
- **Size**: Recommended 800x600 pixels or similar aspect ratio
- **Content**: Screenshot of Excel template showing column headers and sample data
- **Quality**: High resolution for clear preview

## Sample Data to Show

The preview image should display a table with these sample rows:

| NIS   | Nama            | Jenis Kelamin | Kelas    |
|-------|-----------------|---------------|----------|
| 12345 | Ahmad Rizki     | L             | X TKJ 1  |
| 12346 | Siti Nurhaliza  | P             | X TKJ 1  |
| 12347 | Budi Santoso    | L             | X RPL 1  |
| 12348 | Dewi Sartika    | P             | X RPL 1  |

If the image is not available, the system will show a fallback message with a file icon.
